# Spooky

> Spooky experience with spooky fps mechanics

[![First Look](FirstLook.png)]()

**Controls**

- **Movement**: WASD or ZQSD
- **Shoot**: Left Mouse
- **Aim**: Right Mouse
- **Jump**: Space
- **Crouch**: C
- **Sprint**: Shift
- **Pause**: Tab or P

## How to try the game

- Clone or download the Build folder
- Launch `Spooky.exe`
